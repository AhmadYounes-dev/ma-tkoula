Common Questions

How do I play over local WiFi? Make sure all devices are connected to the same WiFi network. One player creates a game room and others join from the main menu using the lobby code provided by the host.

How do I change the dialect? You can select your preferred dialect (English, Lebanese, Syrian, Palestinian, Egyptian, Gulf, or Modern Standard Arabic) from the game settings before starting a round.

How do I restore my in-app purchases? Go to the Premium Content Page and tap "Restore Purchases." Make sure you're signed in with the same Apple ID or Google Account you used for the original purchase.
